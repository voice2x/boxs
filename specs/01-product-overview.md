# 产品总览

## 一、核心定位

**一句话搞定生活记录 — AI 语音是入口，不是功能。**

核心交互：用户说一句话 → AI 自动识别意图 → 分类处理 → 完成。

```
"午饭花了 35 块"          → 记账：餐饮 ¥35
"今天跑了 5 公里"          → 习惯打卡：运动 ✓
"周五下午三点跟老王开会"    → 备忘：周五 15:00 会议
"昨天打车花了 28，奶茶 15"  → 批量记账：交通¥28 + 餐饮¥15
"这周跑步打卡了吗"          → 查询习惯进度
"提醒我明天带身份证"        → 备忘提醒：明天 08:00
```

一句话入口，AI 做 NLU（自然语言理解），自动路由到记账/打卡/备忘三个模块。

## 三、差异化策略

| 维度     | 时光序/探记      | 本产品                               |
|----------|------------------|--------------------------------------|
| 交互方式 | 手动点击、填表   | 语音一句话搞定                       |
| 功能策略 | 大而全（18模块） | 精简三合一，做深做透                 |
| AI 能力  | 无               | NLU + 对话查询 + OCR                 |
| 目标用户 | 所有生活管理需求 | 「懒人」/忙碌人群（不想打开App填表） |
| 学习成本 | 需要熟悉各模块   | 零学习成本，说话就行                 |

核心 Slogan：**「说一句话，记一笔账，打个卡，留个备忘」**

## 四、目标用户

- 不想打开 App 填表的「懒人」
- 日常消费频繁但懒得手动记账的人
- 想追踪习惯但嫌打卡流程繁琐的人
- 需要备忘/提醒但不想打开日历的人

核心价值：**最低摩擦力的生活记录方式。**
