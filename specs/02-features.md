# 功能模块设计

## MVP（最小可行产品）

### 模块 1：AI 语音输入 + 意图识别（核心）

这是整个产品的差异化核心，详细技术方案见 `04-nlu-design.md`。

- 语音转文字（STT）
- NLU 意图分类：记账 / 打卡 / 备忘 / 查询
- 实体提取：金额、类别、时间、商家名
- 支持一句话混合多条指令
- 规则引擎 + LLM 双层处理

### 模块 2：智能记账

- 自动分类（餐饮、交通、购物...共12个分类，详见 `04-nlu-design.md` 意图分类体系）
- 拍照识票/账单 OCR（扫小票自动提取金额+商家）
- 日/周/月统计图表
- 预算设置与超支提醒

### 模块 3：习惯打卡

- 自定义习惯（运动、阅读、早起...）
- 日历热力图展示连续天数
- 打卡提醒推送
- 周/月统计

### 模块 4：备忘/待办

- 时间提醒（定时/重复）
- 智能时间解析（"下周三"、"后天下午"）
- 到期提醒通知
- 简单分类（待办/备忘/想法）

### 模块 5：AI 对话助手

- "这个月餐饮花了多少"
- "我跑步习惯坚持了几天"
- "本周还有什么待办"
- 自然语言查询，无需翻页面
